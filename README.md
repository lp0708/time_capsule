# timecapsule
A virtual time capsule
A simple web-based Virtual Time Capsule built using MERN stack. This project allows users to create digital time capsules, write letters to their future selves, and preserve memories to be revisited later.

Features
User Registration and Login pages
Create and seal virtual time capsules
Dynamic text and background changes using JavaScript
Live date and time display
Dynamic image updates
Dynamic list creation
Mouse event handling
JavaScript animation
Image mapping for interactive navigation
Technologies Used
Frontend: React.js 18, React Router v6, Axios
Backend: Node.js, Express.js
Database: MongoDB Atlas, Mongoose
Auth: JWT (JSON Web Tokens), bcryptjs
